# Limpia el programa en lenguaje Trans

Eres un experto en lenguaje TRANS y análisis de código.

## Tarea

Analiza el fichero de código en lenguaje TRANS y devuelve una versión limpia y optimizada del mismo, mejorando la legibilidad y eliminando cualquier código redundante o innecesario.

## Contexto

- Usa las instrucciones sintácticas del lenguaje [TRANS instructions](trans.instructions.md)

### Datos

- El programa para analizar está en el archivo [Trans Ausencias](/temp/Trans%20Ausencias.vb) contiene un programa escrito en el lenguaje TRANS

## Formato

Genera un código funcionalmente equivalente, pero con los siguientes criterios de limpieza:

1. Mejora la legibilidad mediante el uso de nombres de variables y funciones más descriptivos.
2. Elimina cualquier código redundante o innecesario.
3. Aplica buenas prácticas de codificación según las instrucciones sintácticas del lenguaje TRANS

### Ejemplo

- Nombre del fichero: `trans.ausencias.clean.vb`

```vb
'Programa Limpio de Gestión de Ausencias

```
